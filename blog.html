What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface that represents an HTML or XML document as a structured tree of objects. It allows developers to dynamically access, modify, and manipulate the content, structure, and styles of web pages using scripting languages like JavaScript.

How do you select on element from the Dom?

To select an element from the DOM, use methods like getElementById() for id, getElementsByClassName() for class, getElementsByTagName() for tags, or querySelector() for CSS selectors. For example, document.getElementById("myId") selects an element with the id="myId".

What is event delegation in the context of the Dom ,and why is it useful?

Event delegation is a technique where a single event listener is added to a parent element to manage events for its child elements, improving performance and simplifying dynamic element management.

How do you monipulate an elements attribute and style using the Dom?

To manipulate an element's attribute in the DOM, use setAttribute() or directly modify properties like element.src. For styles, access the style property: element.style.color = "red". You can also toggle classes with classList: element.classList.add("newClass") or remove().
